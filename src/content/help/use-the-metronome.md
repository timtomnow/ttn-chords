The metronome is a standalone practice click with a visual flash. Use it for
warm-ups and tempo practice away from any one song.

## Steps

1. Open **Tools** and tap **Metronome**.
2. Set the tempo:
   - Tap the **−5 / −1 / +1 / +5** buttons or drag the slider, or
   - Tap the big BPM number and type a value (you can clear it; leaving it blank
     keeps the previous tempo).
   - Or tap **Tap** in time to set the tempo by feel.
3. Tap **Start** to run it, **Stop** to halt.
4. Adjust **Meter** (beats per measure, subdivision), **Sound** (voice, downbeat
   accent, volume), and **Light** (flash shape and colours) as needed.

## What you'll see

A beat indicator pulses on each beat, the screen flashes if the light is on, and
your settings are saved for next time (and shared with the metronome inside the
performance screen).
