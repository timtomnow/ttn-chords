A song can hold several difficulty versions (levels 1–5), each with its own
chords and beat timing — for example a simple Level 1 with a few easy chords and a
richer Level 3. Players switch between them while reading or performing.

## Before you start

- Turn on [authoring (admin) mode](/help/turn-on-admin-mode) and open a song's
  editor (see [Create or import a song](/help/create-a-song)).

## Steps

1. In the song editor, find the **Difficulty variants** section.
2. Tap **Add** for a blank version, or **Duplicate** to copy the current one as a
   starting point for a harder arrangement.
3. Select a version to edit it; its sections below now belong to that version.
4. Set the version's **Level (1–5)** and an optional **Label** (e.g. *Beginner*).
5. Tap **Set default** to choose which version opens first. Use **Delete** to
   remove one (a song always keeps at least one).

## What you'll see

Each version is a chip labelled by level/label, with a star on the default. Each
is edited and beat-tagged independently. When a song has more than one version, a
**Difficulty** switcher appears on the song page and in the performance screen.
