A song can have more than one difficulty version — for example a simple
three-chord arrangement and a richer one. When it does, you can switch between
them on the spot, both while reading and while performing.

## Before you start

- The song must have more than one difficulty version. If it only has one, no
  switcher appears. Adding versions is an authoring task — see
  [Add difficulty versions of a song](/help/difficulty-versions).

## Steps

1. Open a song that has multiple versions.
2. On the song page, use the **Difficulty** menu to preview a version.
3. Tap **Perform**, then use the difficulty menu at the top to switch versions
   live.

## What you'll see

The chords (and any beat timing) change to the selected version immediately. Each
version is labelled by its level, e.g. *Level 1* or a custom name like *Simple*.
