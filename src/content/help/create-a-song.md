A song holds its details (title, key, tempo, and so on) and a body of sections
with chords over lyrics. You can start blank or paste existing ChordPro text.

## Before you start

- Turn on [authoring (admin) mode](/help/turn-on-admin-mode).

## Steps

1. Open **Songs**.
2. To start blank, tap **New**. To bring in existing text, tap **Import**, paste
   ChordPro, and import it as a draft.
3. Fill in the details: **Title**, **Artist**, **Key**, **Capo**, **Tempo
   (BPM)**, **Time**, and **Tags**.
4. In each section, type lyrics with inline chords like `[G]`, e.g.
   `[G]Amazing [C]grace`. A live preview shows chords over the words.
5. Use **Add section** to add more, the kind menu to label them (Verse, Chorus,
   …), and the drag handle to reorder.

## What you'll see

Edits save automatically — a **Saved** indicator confirms. Tap **Done** to return
to the song, where you can perform it or tag its beats.
