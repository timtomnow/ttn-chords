**⚠️ Reports are still in development and not yet production-ready. Expect rough
edges and changes; treat output as a draft.**

A report is a page layout you can print — a songbook or handout combining songs,
chord charts, images/logos, rhythm boxes, and text across one or more pages.

## Before you start

- Turn on [authoring (admin) mode](/help/turn-on-admin-mode).

## Steps

1. Open **Reports**.
2. Start one of three ways:
   - **From song** — one page for a chosen song.
   - **From setlist** — one page per song in a setlist.
   - **New** — a blank report.
3. Add blocks from the palette: **Song**, **Text**, **Chord charts**, **Rhythm**,
   **Image / logo**, **Spacer**.
4. Arrange them — stack blocks in the page column, or place floating blocks freely
   and resize them. Add or remove pages as needed.
5. For a Song block, pick the song, the difficulty version, which sections to
   show, and whether to show charts, rhythm, and chords-over-lyrics.

## What you'll see

The editor shows the pages at print size. Song blocks update live if you edit the
underlying song. When it looks right, print it — see
[Print or save a report as PDF](/help/print-a-report).
