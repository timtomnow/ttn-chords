You can run and tune the metronome from inside the performance screen, so you
never have to leave the song to set up a click track.

## Steps

1. Open a song and tap **Perform**.
2. In the bottom bar, tap the metronome button to start or stop the click.
3. Set the tempo:
   - Tap the **−** / **+** buttons to nudge it, or
   - Tap the tempo number and type a new value (you can clear it completely; if
     you leave it blank it keeps the previous value).
4. Tap the sliders/gear button next to the metronome to open **Metronome
   settings**, where you can change:
   - **Sound** — on/off, voice, downbeat accent, and volume.
   - **Light** — the on-screen flash shape and colours.
   - **Highway** — the **Bar number size** (Small, Medium, or Large).

## What you'll see

Changes apply immediately and are saved, so the metronome behaves the same next
time. The bar-number size affects the [Highway view](/help/highway-view).
