A setlist is an ordered group of songs for a class or session. Each song in the
setlist can carry its own transpose, capo, and notes without changing the
original song.

## Before you start

- Building setlists is an authoring task — turn on
  [authoring (admin) mode](/help/turn-on-admin-mode) first.

## Steps

1. Open **Setlists** and tap **New**.
2. Give it a **Name** (and an optional **Description**).
3. Tap **Add songs**, select the songs you want, and confirm.
4. Drag the handle on each row to reorder.
5. For any song, set its **Transpose**, **Capo**, or **Performance notes** — these
   apply only within this setlist.

## What you'll see

The setlist lists its songs in order. The per-song overrides show only here; the
original songs are untouched.
